The BTS  offers an optional wireless remote control kit for beacon operation. This add-on enables remote control, while manual control remains possible without it. The kit streamlines the beacon measurement process and is mandatory for drone-assisted beacon positioning.
The wireless remote control kit consists of a beacon power supply with a wireless module and a remote controller, forming a wireless network. The controller's start/stop buttons allow remote operation of the beacon, with real-time status data (voltage, current, operation status, duration, etc.) displayed on its screen. It also supports software-based control via computer connection.
